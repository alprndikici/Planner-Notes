# 📝 Planner & Notes

Planner & Notes is a clean, minimalist application designed for managing daily, weekly, and long-term plans, taking rich notes, setting reminders, and organizing everything with custom tags. Powered by **Google AI Studio (Gemini API)**.

---

## Features

- 📅 **Smart Calendar & Timeline:** Create and monitor daily schedules, weekly agendas, and long-term targets in a unified dashboard.
- 📓 **Minimalist Note-Taking:** Capture ideas, research, or study materials within a clutter-free, responsive user interface.
- 🏷️ **Category & Tag Management:** Group your notes and tasks with custom tags to filter and locate important details in seconds.
- ⏰ **Integrated Reminders:** Set actionable alerts on your tasks to ensure you never miss a deadline or event.
- 🎨 **Modern Interface:** Beautiful fluid animations and components built using Tailwind CSS v4 and Motion (Framer Motion).

## 🛠️ Tech Stack & Dependencies

This project is built using a modern full-stack JavaScript architecture (React + Express) with strict TypeScript typing:

- **Frontend:** React 19, TypeScript, Tailwind CSS v4, Motion, Lucide React
- **Backend:** Express, Vite Node Middleware
- **Artificial Intelligence:** Google AI Studio Gemini API
- **Build Tools:** Vite ^6.2.3, Esbuild, tsx
