# Planner-Notes
An application, which you can take notes and make daily, weekly and even long time plans. It also supports reminders, tag-based categorization and offers users the option to switch between light and dark modes.

---

## 🚀 Features

- **Intuitive Dashboard:** A clean and welcoming interface displaying quick statistics, recent notes, and upcoming schedules.
- **Dynamic Category Management:** Create, color-code, and manage custom categories (e.g., Work, Personal, Fitness, Studies) to keep notes structured.
- **Rich Note-Taking System:** Create, edit, and organize notes dynamically under specific categories with seamless search capabilities.
- **Integrated Calendar & Planner:** Track deadlines, events, and daily schedules using a built-in interactive planner view.

---

## 🛠️ Tech Stack & Architecture

- **Frontend / UI:** JavaFX - Custom CSS styling for modern flat UI design.
- **Backend Architecture:** Object-Oriented Programming (OOP) utilizing design patterns for separation of concerns (MVC / MVVM architecture).
